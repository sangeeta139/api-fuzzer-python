# API Security Fuzzer (Python)
A lightweight automated API fuzzer for discovering potential security issues like IDOR, broken auth, and improper input handling.


## Features
- Randomized malicious payloads
- Optional JWT auth token
- IDOR detection attempt
- JSON report output
- Simple & extensible Python code
